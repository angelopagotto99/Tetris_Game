# Tetris in C++

A complete Tetris game implementation in C++ with console-based graphics.

## Features
- Full Tetris gameplay with scoring system
- Real-time controls and collision detection  
- Efficient C++ memory management and optimization

## Technologies
- C++11/14
- Standard Template Library (STL)
- Object-Oriented Programming

## Compilation & Execution
```bash
g++ -std=c++11 *.cpp -o tetris
./tetris
